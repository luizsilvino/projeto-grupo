#  MODA | Loja de Roupas

Projeto de uma loja virtual moderna e responsiva, desenvolvida com **HTML, CSS e JavaScript puro**.

---

##  Funcionalidades

*  Listagem de produtos (Masculino e Feminino)
*  Filtro por categoria
*  Carrinho de compras interativo
*  Adicionar produtos ao carrinho
*  Remover produtos do carrinho
*  Cálculo automático do total
*  Simulação de finalização de compra

---

##  Layout

Design minimalista e elegante, inspirado em lojas premium de moda.

* Tipografia moderna (Inter)
* Grid responsivo
* Efeito hover nas imagens
* Interface limpa e intuitiva
